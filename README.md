# 🏥 Hospital Management System

![Django](https://img.shields.io/badge/Django-green) ![Python](https://img.shields.io/badge/Python-blue) ![SQLite](https://img.shields.io/badge/SQLite-purple) ![Razorpay](https://img.shields.io/badge/Razorpay-amber)

A full-stack web application that streamlines hospital operations — patients can book appointments online and pay securely, while doctors manage prescriptions and treatment records through a dedicated portal.

## Features
- 👤 **Patient Portal** — Register, login, book & manage appointments
- 🩺 **Doctor Portal** — View appointments, add prescriptions & treatment notes
- 💳 **Razorpay Payments** — Secure online payment for appointment booking
- 🔒 **Role-based Auth** — Separate access for patients, doctors & admin
- 📅 **Appointment System** — Full CRUD: book, reschedule, cancel
- 📋 **Prescription Management** — Doctors add & update treatment records

## Tech Stack
Python · Django · SQLite · Razorpay API · HTML/CSS · JavaScript

## Getting Started
1. `git clone https://github.com/Hiten1222-re/hospital-management`
2. `pip install -r requirements.txt`
3. Add your Razorpay keys in `settings.py`
4. `python manage.py migrate && python manage.py runserver`
